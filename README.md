# 🎯 InterviewAgent

An AI-powered mock interview platform that helps students and job seekers practice technical and HR interviews in a real interview environment.

🌐 **Live Demo:** [https://interviewagent-8mrs.onrender.com/](https://interviewagent-8mrs.onrender.com/)

---

# 🚀 Features

* 🤖 AI-generated interview questions
* 💬 Real-time interview simulation
* 🔐 Authentication & Authorization
* 📱 Responsive UI
* ⚡ Fast and interactive experience
* 🌐 Full-stack MERN application
* ☁️ Deployed on Render
* 🎯 Technical + HR interview support

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Tailwind CSS
* Axios

## Backend

* Node.js
* Express.js

## Database

* MongoDB

* JWT Authentication

* Render

# ⚙️ Installation

## Clone the repository

```bash
git clone https://github.com/KartikEy4codes/interviewAgent.git
```

## Go to project folder

```bash
cd interviewAgent
```

---

# 🔥 Setup Frontend

```bash
cd client
npm install
npm run dev
```

---

# 🔥 Setup Backend

```bash
cd server
npm install
npm start
```

---

# 🌍 Environment Variables

Create a `.env` file inside the server folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```
